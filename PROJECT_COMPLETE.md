## Getting Started

### Prerequisites
- Docker Desktop
- Python 3.10+
- Node.js 18+
- Git

### Quick Start

1. **Clone Repository**
   ```bash
   git clone <repository-url>
   cd AI-Powered-Self_Optimizing_Data_Warehouse
   ```

2. **Start Services**
   ```bash
   make start-all
   ```

3. **Access Services**
   - Dashboard: http://localhost:3000
   - API Gateway: http://localhost:8000
   - Grafana: http://localhost:3001
   - Prometheus: http://localhost:9090

4. **Run Tests**
   ```bash
   make test
   ```
